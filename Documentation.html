Documentation API

La 1ère API que nous avons utlisé est Email Verifier:


Cette API à pour but de vérifier l'existence du nom de domaine de l'expéditeur et de celui qui reçoit le mail.

La 2éme API que nous avons utilisé est le Grammar Bot :


Cette API à pour but de détécter puis de corriger les fautes d'orthographe présentes dans le contenu du mail.

Nous proposons aux utilisateurs:


Une API qui affiche un tableau avec le nombre d'occurence des domaines les plus fréquents
Ou la moyennes des erreurs en fonction du nom de domaines


Documentation de notre API


Voici quelques exemples d'appel:


APPEL : serveur.php

ici il n'y a pas de filtre -------retourne les domaines et les occurrences qui leurs correspondent

[{"domain":"gmail","occurrence":40},{"domain":"yahoo","occurrence":50}]

APPEL : serveur.php?domain=gmail

ici on filtre par domaine en précisant gmail en paramètre
------retourne le nombres d'erreur en fonction du nom de domaine renseigné

{domain : gmail, occurences : 200}

APPEL : serveur.php?mistakes=true

ici le filtre va donner la moyenne d'erreurs dans tous les domaines enregistrés
------retourne la moyenne du nombre d'erreur de tous les domaines enregistrés

[{"domain":"gmail","averageMistakesPerMail":5}][{"domain":"yahoo","averageMistakesPerMail":6}]

APPEL : serveur.php?domain=gmail&mistakes=true

ici on filtre par domaine ------retourne la moyenne du nombre d'erreur en fonction du nom de domaine renseigné

{domain : gmail, occurences : 200, averageMistakesPerMail : 20}
